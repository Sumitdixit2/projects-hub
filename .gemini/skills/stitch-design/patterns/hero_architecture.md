# Hero Architecture

The hero section must instantly communicate value and establish the aesthetic baseline.

## Pattern 1: The Centered Cinematic
- **Layout:** Centered content. 
- **Typography:** Massive, gradient or high-contrast H1.
- **Visuals:** A massive, high-fidelity UI mockup or 3D element sits *below* the fold, fading upwards into the text.
- **Best for:** Consumer SaaS, Developer Tools.

## Pattern 2: The Split Asymmetric
- **Layout:** 50/50 horizontal split.
- **Typography:** Left-aligned, extremely dense, high contrast.
- **Visuals:** An interactive element, code block, or telemetry graph pinned to the right side.
- **Best for:** B2B SaaS, Technical Infrastructure.

## Pattern 3: The Terminal / Minimal
- **Layout:** Constrained, left-aligned prose column (`max-w-prose`).
- **Typography:** Monospace or clean sans-serif. No massive sizes.
- **Visuals:** Almost none. Relies entirely on typographic hierarchy and macro-whitespace.
- **Best for:** Portfolios, Deep-tech products.
