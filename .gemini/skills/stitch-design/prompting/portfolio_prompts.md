# Portfolio Prompts

Generating portfolios requires a focus on showcasing work without the interface overpowering the content.

## 1. The Minimalist Architect
> "Design a personal portfolio hero section. The vibe is extreme minimalism, like an art gallery or architectural firm. Use a stark white background and a serif font for the main headline. The headline should be massive and offset to the right. On the left, a very small, meticulously aligned paragraph describing the person's role. Below the fold, use a masonry layout for project images with no visible borders, only appearing with a subtle, slow (500ms) fade-in on hover. Keep the navigation to just 3 simple text links at the absolute top corners."

## 2. The Creative Technologist
> "Design a portfolio for a creative developer. Dark mode. The aesthetic is heavily inspired by Apple's technical landing pages. Centralized, large gradient text for the headline. Below the headline, an interactive WebGL or abstract 3D component (represent this as a placeholder box with a subtle glass effect). Use a bento grid below this to show 'Skills', 'Recent Work', and 'Currently Learning'. Every card should have a 1px inner border of `rgba(255,255,255,0.1)`."

## 3. The UX Engineer
> "Design a case study page for a UX engineer portfolio. The layout must prioritize reading experience. A narrow, left-aligned prose column (`max-w-2xl`). To the right of the prose column, use a sticky sidebar that contains a table of contents and a metadata block (Role, Timeline, Stack). Use high contrast typography. Include full-bleed image blocks that break out of the prose container to show high-fidelity mockups."
