# Elite Hero Prompts

Use these prompts in Stitch to generate high-end hero sections.

## 1. The Stripe-Esque Developer Hero
> "Design the hero section for a modern developer API tool. Dark mode. The layout should be split 50/50. On the left, a highly dense, left-aligned typographic block: an `h1` in a tight sans-serif, a secondary paragraph in `text-zinc-400`, and two buttons (one solid primary, one ghost). On the right, a floating, glassmorphic code editor window showing a syntax-highlighted JSON snippet. The background should be a deep off-black with an extremely subtle, dark radial gradient in the top right corner to provide depth. Do not use any illustrations. Use a 1px drafting border for the code window."

## 2. The Linear-Esque Tooling Hero
> "Design a minimalist SaaS hero section. Background is pure black (`#000000`). The typography is the star: center-aligned, massive, tightly tracked heading. Below it, a thin, subtle pill-shaped badge with a glowing 'New Feature' dot. Below the text, show the top 20% of the application dashboard rising up from the bottom of the screen, fading into view with a CSS mask-image linear gradient so it looks like it's emerging from the darkness. Use a 1px border on the dashboard edge. No gradients in the text."

## 3. The Hardware/Cyberpunk Aesthetic Hero
> "Design a brutalist, terminal-inspired hero section. Use a monospace font exclusively. The color palette is dark gray and high-contrast neon orange. The layout should feel like a dense heads-up display (HUD). Include a left sidebar with telemetry data (numbers and tiny sparkline graphs). The main central area should have a large, uppercase headline. Use sharp corners (0px border-radius) everywhere. Include subtle horizontal grid lines across the background."
