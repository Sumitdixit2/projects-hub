# SaaS Product Prompts

SaaS interfaces must balance marketing appeal with apparent utility.

## 1. The B2B Landing Page
> "Design a landing page for a B2B SaaS product. Use a light mode aesthetic. The hero section should feature a clear 'H1' value proposition, a supportive subtext, an email input field, and a primary 'Get Started' button. Immediately below the hero text, show a highly detailed, realistic browser mockup of the app dashboard. The background should be a subtle, off-white mesh gradient to give the page texture without distracting from the UI mockup."

## 2. The Pricing Page
> "Design a pricing tier section for a SaaS app. Use three cards. The middle card ('Pro Tier') should be visually elevated: slightly taller, with a distinct colored top border (e.g., bright blue) and a subtle drop shadow, while the other two cards are flat with just a 1px gray border. Each card must clearly list the price in a large font, followed by a dense, left-aligned list of features with checkmark icons. Include an interactive toggle at the top for 'Monthly / Annually'."

## 3. The Feature Bento
> "Design a 'Features' section for a SaaS landing page using a bento grid. Instead of text-heavy cards, each card should be an 'interactive' mini-demonstration of a feature. Use a 3x2 grid. The top-left card spans 2 columns. Cards must have a dark mode aesthetic with `bg-zinc-900` and a `bg-zinc-950` canvas. Provide a subtle gradient glow in the corner of each card behind the illustration."
