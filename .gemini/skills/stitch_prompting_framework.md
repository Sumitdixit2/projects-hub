# The Stitch Prompting Framework

To generate elite interfaces, use the 5-part structure to provide explicit constraints.

## 1. Context & Persona
Define the vibe (e.g., "Developer-focused infrastructure dashboard").

## 2. Layout Geometry
Dictate spatial arrangement (e.g., "Asymmetrical bento grid").

## 3. Visual Language
Specify color, borders, and depth (e.g., "Monochromatic zinc, 1px borders, no shadows").

## 4. Typographic Hierarchy
Constrain text styles (e.g., "Monospace for telemetry numbers").

## 5. Interaction Details
Define motion (e.g., "Pill-shaped geometric buttons, crisp transitions").

## Negative Prompting (Crucial)
- "DO NOT use large corner radii (4px max)."
- "DO NOT use heavy drop shadows."
- "DO NOT use illustrative graphics; use code/geometric shapes."
